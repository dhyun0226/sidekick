## 1. 시작하기

### 1.1 로그인
- 소셜 로그인 (Google 등) 지원
- 처음 로그인 시 온보딩 화면으로 자동 이동

### 1.2 온보딩
- 닉네임 설정
- 프로필 사진 업로드 (선택)

### 1.3 그룹 참여 방법

**방법 1: 초대 코드로 참여**
1. 메인 화면 우측 상단 🔑 버튼 클릭
2. 8자리 초대 코드 입력
3. "참여하기" 버튼 클릭

**방법 2: 초대 링크로 참여**
- 친구가 보낸 초대 링크 클릭 → 자동으로 그룹 참여

## 2. 메인 화면

### 2.1 화면 구성
```
┌─────────────────────────────┐
│  👤 프로필   🔑   🔔        │
│  My Library                 │
│  오늘의 독서를 시작해볼까요?   │
├─────────────────────────────┤
│  🔥 지금 읽고 있어요          │
│  [그룹 카드 1]              │
│  [그룹 카드 2]              │
├─────────────────────────────┤
│  💤 잠시 쉬고 있어요          │
│  [그룹 카드 3]              │
├─────────────────────────────┤
│           [➕ 버튼]          │
└─────────────────────────────┘
```

### 2.2 그룹 상태

**🔥 지금 읽고 있어요**
- 현재 읽는 중인 책이 있는 그룹
- 카드에 표시되는 정보:
  - 그룹 이름
  - 책 표지 (배경 블러 처리)
  - 책 제목 & 저자
  - D-Day (남은 일수)
  - 멤버 수
  - 독서 기간
  - 내 진행도 (%)

**💤 잠시 쉬고 있어요**
- 현재 읽는 책이 없는 그룹
- 카드에 표시되는 정보:
  - 그룹 이름
  - 멤버 수

### 2.3 주요 기능

#### ➕ 새 그룹 만들기
1. 우측 하단 ➕ 버튼 클릭 (그룹이 있을 때)
   - 또는 빈 화면에서 "새 그룹 만들기" 버튼 클릭
2. 그룹 이름 입력
3. 생성 완료

**제한 사항:**
- 무료 플랜: 최대 2개 그룹
- 프리미엄: 무제한

#### 🔑 초대 코드로 입장
1. 우측 상단 🔑 버튼 클릭
2. 8자리 초대 코드 입력
3. "참여하기" 버튼 클릭

#### 🔔 알림 센터
- 새 댓글 알림
- 멤버 참여 알림
- 책 완주 알림
- 반응(좋아요) 알림

## 3. 그룹 독서 화면

### 3.1 화면 구성
```
┌─────────────────────────────┐
│  ← 뒤로    책 제목      ☰   │
├─────────────────────────────┤
│      📖 책 히어로 섹션        │
│   (책 표지, 제목, D-Day)     │
├─────────────────────────────┤
│                             │
│      💬 타임라인            │
│    (댓글 그룹 목록)          │
│                             │
├─────────────────────────────┤
│    🎚️ 슬라이더              │
│    [현재 챕터명]    ✍️       │
└─────────────────────────────┘
```

### 3.2 슬라이더 (SmartSlider)

#### 📍 기본 구조
- **챕터 배경**: 각 챕터별로 배경색 구분 (현재 챕터 강조)
- **눈금**: 5% 간격 눈금 표시 (10% 단위는 길게)
- **진행도 바**: 라임색 바로 현재 진행도 표시
- **슬라이더 핸들**: 드래그 가능한 원형 핸들

#### 🎯 슬라이더 모드

**1. 읽는 중 - 진행도 추적 모드**
- 슬라이더 드래그 → 진행도 저장 (2초 후)
- 내 아바타가 슬라이더를 따라 움직임
- 현재 위치보다 뒤의 댓글만 볼 수 있음

**2. 완독 후 - 네비게이션 모드**
- 슬라이더 드래그 → 진행도 저장 안 됨
- 내 아바타는 100% 위치에 고정
- 모든 댓글 열람 가능 (스포일러 잠금 해제)

#### 🎨 슬라이더 표시 요소
- **멤버 아바타**: 다른 멤버들의 진행도 위치에 표시 (최대 3명)
- **드래그 시 툴팁**: 진행도%, 페이지, 챕터명 표시
- **하단 표시**: 현재 챕터명
- **우측 버튼**: ✍️ 댓글 작성 버튼

#### 📱 슬라이더 사용법
1. 슬라이더 핸들을 드래그
2. 원하는 위치에서 손을 떼면 자동 저장 (2초 후)
3. 드래그 중 타임라인이 실시간으로 스크롤됨

### 3.3 타임라인 (댓글 목록)

#### 📝 댓글 그룹
- **위치별 그룹핑**: 같은 위치(%)의 댓글들을 하나의 그룹으로 묶음
- **인용구 표시**: 그룹 내 인용 텍스트가 있으면 맨 위에 표시
- **미리보기**: 각 그룹에서 상위 2개 댓글만 표시
- **더보기 버튼**: 3개 이상일 경우 "더보기" 버튼 표시

#### ✍️ 댓글 작성
1. **방법 1**: 슬라이더 하단 ✍️ 버튼 클릭
2. **방법 2**: 타임라인에서 특정 위치의 댓글 그룹 클릭 (계획됨)

**작성 화면:**
- **인용 텍스트**: 선택사항, 책 내용을 직접 입력
- **댓글 내용**: 필수, 최대 500자
- **위치**: 현재 슬라이더 위치가 자동 저장

#### ❤️ 상호작용
- **좋아요**: 하트 버튼 클릭
- **답글**: 답글 버튼 클릭 → 답글 작성
- **수정**: 본인 댓글만 수정 가능 (수정 버튼)
- **삭제**: 본인 댓글만 삭제 가능 (삭제 버튼)

#### 🔒 스포일러 보호
- **미완독 시**: 내 진행도보다 앞의 댓글은 블러 처리
- **완독 후**: 모든 댓글 열람 가능

#### 📜 무한 스크롤
- 타임라인 하단에 도달하면 자동으로 추가 댓글 로드
- "모든 댓글을 불러왔습니다" 메시지로 끝 표시

### 3.4 완독 처리

#### ✅ 완독하기
1. 우측 상단 ☰ 버튼 클릭 (드로어 열기)
2. "정보" 탭 선택
3. 해당 책 카드의 ⋮ 메뉴 클릭
4. "완독 처리" 선택
5. 리뷰 작성 모달 자동 표시 (선택)

**완독 후 변화:**
- ✅ 슬라이더가 네비게이션 모드로 전환 (진행도 저장 안 됨)
- ✅ 모든 댓글 열람 가능 (스포일러 블러 해제)
- ✅ 멤버 탭에 완독 날짜 표시
- ✅ 프로필 서재에 책 추가

#### ↩️ 완독 취소
1. 드로어 메뉴 → 정보 탭
2. 해당 책 카드의 ⋮ 메뉴 → "완독 취소"
3. 진행도는 100%로 유지되지만, 완독 상태 해제

### 3.5 리뷰 작성

#### ⭐ 리뷰 작성/수정
1. 드로어 메뉴 → 정보 탭 또는 책장 탭
2. 해당 책 카드의 ⋮ 메뉴 → "리뷰 작성" (또는 "리뷰 수정")
3. 별점 선택 (0.5점 단위, 0~5점)
4. 리뷰 내용 작성
5. "저장" 버튼 클릭

**리뷰 확인:**
- 드로어 메뉴 → 책장 탭 → 해당 책 카드의 "⭐ X개 리뷰" 버튼 클릭
- 모든 멤버의 리뷰 확인 가능

## 4. 드로어 메뉴

### 4.1 열기/닫기
- 우측 상단 ☰ 버튼 클릭
- 우측에서 드로어 슬라이드

### 4.2 드로어 탭 구조

드로어는 **4개 탭**으로 구성됩니다:
- **정보**: 읽는 중인 책 목록 + 목차
- **멤버**: 멤버 목록 + 진행도
- **책장**: 완주한 책 목록
- **설정**: 그룹 설정 + 책 추가

---

### 📖 정보 탭

#### 읽는 중인 책 목록
- 현재 읽는 중인 책들 표시
- 각 책 카드에 표시:
  - 책 표지, 제목, 저자
  - 독서 기간 (시작일 ~ 종료일)
  - D-Day (목표일까지 남은 일수)
  - 회차 표시 (같은 책을 여러 번 읽은 경우)

#### 책 선택
- 책 카드 클릭 → 해당 책의 타임라인으로 전환

#### 목차 (선택된 책)
- 선택된 책의 목차 표시
- 챕터 클릭 → 해당 위치로 이동 (슬라이더 + 타임라인 자동 이동)
- 현재 읽는 챕터 하이라이트 표시

#### 책 메뉴 (⋮)

**모든 사용자:**
- **리뷰 작성/수정**: 내 리뷰 작성 또는 수정
- **완독 처리**: 책을 다 읽었을 때 완독 표시 (리뷰 작성 모달 자동 표시)
- **완독 취소**: 완독 상태 해제

**관리자 전용:**
- **기간 수정**: 독서 시작일/종료일 변경
- **목차 수정**: 챕터 구성 및 총 페이지 수 변경
- **완주 처리**: 그룹 전체 책 완료 처리 (책장으로 이동)
- **책 삭제**: 그룹에서 책 제거 (모든 댓글/리뷰 삭제)

---

### 👥 멤버 탭

#### 멤버 목록 정렬

**읽는 중인 책일 때 (진행도 순):**
- 진행도 높은 순으로 정렬
- 각 멤버 정보:
  - 아바타
  - 닉네임
  - 진행도 (%)
  - 마지막 활동 시간 (예: "2일전")
  - 완독 배지 (✅ 완독한 경우)
  - 페이스메이커 배지 (🏃 1등)

**완주한 책일 때 (완독 순):**
- 빨리 완독한 순으로 정렬
- 완독 날짜 표시 (예: "25.01.15 완독")

#### 멤버 메뉴 (관리자 전용)

멤버 카드의 ⋮ 버튼 클릭:
- **관리자 승격**: 일반 멤버를 관리자로 승격
- **강퇴**: 멤버를 그룹에서 제거

---

### 📚 책장 탭

#### 완주한 책 목록
- 그룹에서 완주 처리된 책 목록
- 표시 정보:
  - 책 표지, 제목, 저자
  - 완주일 (예: "25.01.15 완주")
  - 회차 (같은 책 여러 번 읽은 경우)
  - 리뷰 개수 (⭐ X개 리뷰)

#### 내가 완독하지 않은 책
- 잠금 아이콘 (🔒) 표시
- 클릭 불가 (스포일러 방지)
- 완독 처리 후 열람 가능

#### 책 메뉴 (⋮)

**모든 사용자:**
- **리뷰 작성/수정**: 내 리뷰 작성 또는 수정
- **리뷰 모두 보기** (⭐ X개 리뷰 버튼): 모든 멤버의 리뷰 확인
- **완독 처리**: 아직 완독하지 않은 경우
- **완독 취소**: 이미 완독한 경우

**관리자 전용:**
- **다시 읽기**: 완주한 책을 다시 읽는 중으로 변경 (정보 탭으로 이동)
- **완주 날짜 수정**: 완주 날짜 변경
- **책 삭제**: 그룹에서 완전히 제거 (모든 댓글/리뷰 삭제)

#### 프리미엄 제한
- **무료 플랜**: 최근 10권까지만 표시
- **프리미엄**: 전체 기록 열람

---

### ⚙️ 설정 탭

#### 그룹 설정

**그룹 이름 변경 (관리자 전용):**
1. 그룹 이름 입력란에 새 이름 입력
2. "저장" 버튼 클릭

**초대 관리:**
- **초대 코드 복사**: 8자리 코드 클립보드 복사
- **초대 링크 복사**: 전체 URL 클립보드 복사
- **초대 코드 재생성** (관리자 전용): 기존 코드 무효화, 새 코드 생성

#### 책 추가 (관리자 전용)

1. "새 책 시작하기" 버튼 클릭
2. ISBN으로 검색 또는 직접 입력
3. 목차 입력 (선택)
4. 독서 기간 설정 (시작일 ~ 종료일)
5. "추가" 버튼 클릭

**제한 사항:**
- 무료 플랜: 그룹당 최대 10권 (읽는 중 + 완주 합산)
- 프리미엄: 무제한

#### 그룹 나가기

1. "그룹 나가기" 버튼 클릭
2. 확인 모달에서 "나가기" 클릭
3. 나가면 내 진행도, 댓글, 리뷰 모두 삭제됨 (복구 불가)

**제한사항:**
- 관리자는 다른 관리자가 있거나 멤버가 없을 때만 나갈 수 있음

#### 그룹 삭제 (관리자 전용)

1. "그룹 삭제" 버튼 클릭
2. 확인 모달에서 "다음" 클릭
3. 그룹 이름 정확히 입력
4. "삭제" 버튼 클릭
5. 그룹 및 모든 데이터 영구 삭제

## 5. 프로필 화면

### 5.1 화면 구성
```
┌─────────────────────────────┐
│  ← 뒤로              ⚙️     │
│                             │
│  👤 아바타                  │
│  닉네임                     │
│  오늘도 즐거운 독서 되세요!   │
│                             │
│  [완독] [기록] [연속] [그룹] │
│   12     34     5     3    │
├─────────────────────────────┤
│      💎 구독 정보 카드       │
├─────────────────────────────┤
│  [서재] [기록] [분석]        │
├─────────────────────────────┤
│                             │
│      탭별 콘텐츠             │
│                             │
└─────────────────────────────┘
```

### 5.2 상단 통계

클릭 가능한 4개 통계:
- **완독**: 총 완독한 책 수 (클릭 → 서재 탭)
- **기록**: 총 댓글+리뷰 수 (클릭 → 기록 탭)
- **연속**: 현재 연속 독서 일수 (클릭 → 분석 탭)
- **그룹**: 참여 중인 그룹 수 (클릭 → 메인 화면)

### 5.3 구독 정보 카드

**프리미엄 사용자:**
- 그라데이션 배경 카드
- 구독 기간 (만료일)
- 자동 갱신 상태
- "구독 관리" 버튼 (→ 구독 관리 화면)

**무료 사용자:**
- 점선 테두리 카드

### 📚 서재 탭

#### 책 그리드
- 완독한 모든 책을 4열 그리드로 표시
- 각 책 카드에 표시:
  - 책 표지
  - 내 별점 (⭐ 또는 ──)
  - 완독 월일 (예: "1.15")

#### 책 상세 모달

책 클릭 시 모달 표시:
- **상단**: 책 표지, 제목, 저자, 내 별점, 완독 날짜
- **3개 탭**:
  - **전체**: 모든 기록 (댓글 + 리뷰) 시간순
  - **리뷰**: 내 리뷰만
  - **코멘트**: 내 댓글만

**기록 클릭:**
- ✅ 완독한 책: 해당 그룹의 타임라인으로 이동 (위치 + 하이라이트)
- 🔒 완독 안 한 책: 클릭 불가 (스포일러 방지, 흐리게 표시)

### 📝 기록 탭

#### 타임라인
- 모든 그룹의 댓글 + 리뷰를 최신순으로 표시
- 각 기록 카드에 표시:
  - 책 제목
  - 그룹 이름
  - 작성 시간 (예: "2일전")
  - **댓글**: 진행도 (예: "45%"), 인용구, 내용
  - **리뷰**: 별점, 내용

**기록 클릭:**
- ✅ 완독한 책: 해당 그룹의 타임라인으로 이동
- 🔒 완독 안 한 책: 클릭 불가 (흐리게 표시)

## 6. 구독 관리

### 6.1 구독 플랜

#### 🆓 무료 플랜
- 그룹 생성: 최대 2개
- 그룹당 책: 최대 10권 (읽는 중 + 완주 합산)
- 책장 열람: 최근 10권
- 분석 기능: 제한됨

#### 👑 프리미엄 플랜 (월 ₩2,500)
- ✅ 무제한 그룹 생성
- ✅ 무제한 책 추가
- ✅ 전체 책장 열람
- ✅ 고급 독서 분석
  - 연간 목표 설정
  - 독서 히트맵
  - 월별 상세 통계
  - 연속 독서 일수

### 6.2 구독하기

**프로필 화면에서:**
1. "프리미엄으로 업그레이드" 카드 클릭
2. 플랜 선택 (월간/연간)
3. 토스 페이먼츠 결제창
4. 결제 완료

**연간 구독 혜택:**
- 할인 적용 (월 ₩2,500 → 연 ₩19,000, 약 37% 할인)

### 6.3 구독 관리

**프리미엄 사용자 (프로필 화면):**

   - 만료일

   - 플랜 변경
   - 구독 취소

## 7. 자주 묻는 질문 (FAQ)

### Q1. 그룹에 멤버를 초대하려면?
**A:** 드로어 메뉴 → 설정 탭 → "초대 코드 복사" 또는 "초대 링크 복사"를 눌러 친구에게 공유하세요.

### Q2. 완독 처리와 완주 처리의 차이는?
**A:**
- **완독 처리**: 개인이 책을 다 읽었을 때 (모든 멤버 가능)
- **완주 처리**: 그룹 전체가 책을 끝냈을 때 (관리자만 가능, 책장으로 이동)

### Q3. 완독 후에도 댓글을 쓸 수 있나요?
**A:** 네! 완독 후에는 슬라이더가 네비게이션 모드로 전환되어 원하는 위치에서 자유롭게 댓글을 작성할 수 있습니다. 진행도는 저장되지 않습니다.

### Q4. 진행도가 자동으로 저장되지 않아요.
**A:** 완독 처리한 책은 진행도가 저장되지 않습니다. 완독 후 슬라이더는 네비게이션 도구로만 사용됩니다.

### Q5. 무료 플랜에서 그룹을 3개 이상 만들 수 없어요.
**A:** 무료 플랜은 최대 2개 그룹까지 생성할 수 있습니다. 더 많은 그룹을 만들려면 프리미엄 구독이 필요합니다.

### Q6. 인용구를 어떻게 추가하나요?
**A:** 댓글 작성 시 "인용 텍스트" 입력란에 직접 입력하세요. 책에서 복사한 문구를 붙여넣을 수 있습니다.

### Q7. 프로필의 기록을 클릭해도 이동하지 않아요.
**A:** 완독하지 않은 책의 기록은 클릭할 수 없습니다. 스포일러 방지를 위해 완독 후에만 기록으로 이동할 수 있습니다.

### Q8. 책을 삭제하면 모든 데이터가 사라지나요?
**A:** 네, 책을 삭제하면 해당 책의 모든 댓글, 리뷰, 진행도가 영구적으로 삭제됩니다. 신중하게 결정하세요.

### Q9. 그룹에서 나가면 내 데이터는 어떻게 되나요?
**A:** 그룹을 나가면 해당 그룹에서의 모든 진행도, 댓글, 리뷰가 삭제됩니다. 복구할 수 없으니 신중하게 결정하세요.

### Q10. 스포일러는 어떻게 방지되나요?
**A:**
- 타임라인에서 내 진행도보다 앞의 댓글은 자동으로 블러 처리됩니다.
- 완독 후에는 모든 댓글을 볼 수 있습니다.
- 프로필의 완독하지 않은 책 기록은 클릭이 불가능합니다.

## 8. 팁과 요령

### 💡 효율적인 독서 관리

#### 📅 목표 기간 설정
- 책을 추가할 때 현실적인 목표 기간을 설정하세요
- D-Day를 활용해 독서 속도를 조절하세요

#### 📝 댓글 활용
- 인상 깊은 구절은 인용구와 함께 댓글로 남기세요
- 다른 멤버의 댓글에 답글을 달며 토론하세요

#### ⭐ 리뷰 작성
- 완독 후 바로 리뷰를 작성하면 생각이 선명합니다
- 별점과 함께 짧은 소감을 남겨보세요

### 🎯 그룹 운영 팁 (관리자)

#### 📚 책 선정
- 멤버들과 함께 책을 정하면 참여도가 높아집니다
- 목표 기간을 넉넉하게 설정하세요 (2-4주 권장)

#### 👥 멤버 관리
- 초대 코드를 주기적으로 재생성해 보안을 유지하세요
- 비활동 멤버는 개인적으로 연락해보세요

#### 🎉 완주 처리
- 모든 멤버가 완독할 때까지 기다려주세요
- 완주 후 다음 책을 빠르게 시작하면 연속성이 유지됩니다

### 🌟 독서 습관 만들기

#### 📊 통계 활용 (프리미엄)
- 연간 목표를 설정하고 매달 확인하세요
- 히트맵으로 독서 패턴을 파악하세요
- 연속 독서 일수를 늘려가며 동기부여하세요

#### 🔔 알림 설정
- "내 글에 달린 답글" 알림을 켜두면 토론에 빠르게 참여할 수 있습니다
- "새 책 추가 알림"으로 다음 책 시작 시점을 놓치지 마세요

## 9. 문제 해결

### 🔧 일반적인 문제

#### 진행도가 저장되지 않아요
1. 인터넷 연결을 확인하세요
2. 완독 처리한 책은 진행도가 저장되지 않습니다 (네비게이션 모드)
3. 페이지를 새로고침하고 다시 시도하세요

#### 댓글을 수정/삭제할 수 없어요
- 본인이 작성한 댓글만 수정/삭제할 수 있습니다
- 다른 멤버의 댓글은 수정할 수 없습니다

#### 책이 검색되지 않아요
- ISBN을 정확히 입력했는지 확인하세요
- ISBN이 없는 경우 직접 입력 모드를 사용하세요

#### 그룹 초대 링크가 작동하지 않아요
- 초대 코드가 재생성되었을 수 있습니다
- 관리자에게 새 초대 링크를 요청하세요

#### 프리미엄 기능이 잠겨 있어요
- 구독 상태를 확인하세요 (프로필 → 구독 정보)
- 결제가 완료되지 않았을 수 있습니다
- 구독이 만료되었는지 확인하세요
